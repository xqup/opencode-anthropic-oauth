## 📌 What this app does

opencode-anthropic-oauth is an OpenCode plugin that helps you sign in to Anthropic Claude Pro or Max with OAuth.

It is made for people who want to use Claude access without setting up Claude Code. You download the app, run it on Windows, and follow the login steps.

This tool is meant to:
- connect OpenCode with your Anthropic account
- use OAuth login instead of manual token setup
- make sign-in easier for Claude Pro and Max users
- keep the setup process short

## 🪟 Windows requirements

## 🔐 How the login works

This plugin uses Anthropic OAuth login.

That means:
- you sign in with your Anthropic account
- the app opens your browser for login
- Anthropic handles the account sign-in
- the plugin connects the login to OpenCode

You do not need to copy a long token by hand. You use the normal web login flow.

## 🧰 Troubleshooting

### The app will not open
- Make sure the download finished
- Check that you extracted the file if it came in a .zip
- Right-click the app and choose Open
- If Windows blocks it, choose More info and then Run anyway

### The browser does not open
- Check your internet connection
- Make sure a browser is installed
- Try setting your browser as the default browser
- Run the app again

### Sign-in fails
- Check that you used the correct Anthropic account
- Try logging out of Anthropic in your browser first
- Open a fresh browser window and try again
- Make sure the release version is the latest one

### The app closes right away
- Run it again and wait for any browser window to appear
- Check whether Windows Security blocked it
- Download the latest release file again if the file looks damaged

### OpenCode does not show the connection
- Restart OpenCode
- Run the plugin again
- Confirm that the login finished in your browser
- Sign in again if your session expired
